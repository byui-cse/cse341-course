Teaching Notes

Important topics to teach your students this lesson. These topics will come primarily from the reading and should contain information that will help them with their assignments.

MVC Pattern

Successfully setting up the course project will help the students to have a better understanding of course principles. By understanding the logic behind the MVC Pattern students will more easily understand the principles of the course.

  • Model: Explain what is a model and how will it be used in the context of a json file (i.e the prove assignment)

  • View: Why EJS? What are some of the benefits of the various rendering services that Max discusses (EJS, Handlebars, PUG)

  • Controller: Help the students to distinguish the difference between the controller and the model. Show how controller functions can be used together with routing to create server logic

JSON

  • Explain the reason for using JSON data and simply relate it to later use of a NoSQL database

  • Help the students to understand how data retrieval relates to and is used in the MVC pattern

Routing

  • Teach the difference between GET and POST

  • Help the students understand how these requests are handled in regards to routing

  • Dynamic routing will be an important concept as it will help the students when displaying product details in their ecommerce projects
