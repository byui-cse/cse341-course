Templating Engines

Students will be working with Template engines this entire semester. The majority of it will be in EJS but they can use a different engine if they like. Help them understand the following concepts:

  • Template engines are simply a way to create what is essentially a dynamic html document.

  • There are many different template engines. Some of the more common template engines include Pug, Mustache, EJS, Handlebars, DoT.js, NunJucks, Underscore, ECT, Hogan, Dust.

  • The Academind course will show many examples in Pug and Handlebars, but the majority of the videos and cse341 as a whole will be using EJS.

Working with EJS

Why are we going to focus on EJS?

  • Use plain JavaScript

  • Speedy execution

  • Fast development time

  • Easy debugging

  • Simple syntax

  • Active development

Show them some examples of how ejs looks. You could use the project template, or other code. But help them understand that that can write plain html, and then insert ejs tags anywhere they want/need to execute JavaScript.
