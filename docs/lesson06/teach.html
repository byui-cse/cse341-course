Overview

Your team activity this week will be very flexible. Your goal, is to create a simple application to get practice working with authentication. You may choose anything that does this as long as it is not the exact same code from your reading. You can and should use the videos and code that you wrote from the reading to help you during this assignment.

Instructions

Create an application (or webpage)

  1. Incorporate Authentication

  2. Incorporate Validation

  3. Incorporate Error Handling

This assignment is intended to be open-ended. Working through a task like this helps to build certain skills that other, more rigid assignments cannot build. Your ability to take vague requirements and build something useful is an extremely valuable skill.
