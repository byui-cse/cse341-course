Final Exam Study Guide

Overview

The final exam is cumulative, meaning it will include concepts taught throughout the entire semester. The following are concepts that are covered in lessons 8-12. You should be familiar and comfortable with these concepts as well as all the concepts listed in the Midterm Study Guide prior to taking the final exam in lesson 13.

Topics that may be covered in the exam

  • Week 8

  • File Upload

  • Pagination

  • Payment

  • Weeks 9 & 10

  • REST APIs

  • HTTP Verbs

  • JWT

  • Week 11

  • Asynchronous code

  • Websocket protocol

  • GraphQL vs REST

  • Week 12

  • Deployment

  • Production and hosting

  • Run and define tests (Mocha and Chai)

For a description of exam logistics (rules, question types, time limit, due date, etc.), please reference the exam in I-learn.
