Overview

This lesson's activity will focus on the potential uses of socket.io in your application.

Assignment

  1. Using your code from the W10 activity, implement socket.io on both the client JS and server to emit an update to the list from one client to all other clients. This means that if a the client on one browser adds a new name, all other cleints should update the list with that new name as well.

Hints:

  • You should not have to add many lines of code to implement socket.io

  • Make sure socket.io and socket.io-client are installed on your application

  • On your EJS page, consider including a script tag with src "/socket.io/socket.io.js" above your client-side script tag

  • A cheatsheet for emit methods can be found here: Emit Cheatsheet (socket.io)

  • It's a good idea to read documentation or guides for any web/software technology you may be unfamiliar with. This is a good starting point: Get Started (socket.io)
