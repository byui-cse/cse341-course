Summary

Welcome to Lesson Eight of CSE 341! Our course will take a large turn this lesson.

Up to this point, we have been developing Node.js applications that implement serverside rendering...basically, we've been rendering html templates, or ejs with our Node.js, effectively building fullstack applications in Node. That is all about to change. What is even more widely used in industry today is something called REST APIs.

The concept here is that instead of a server returning html along with data, the server just returns data...thus decoupling the frontend and the backed, giving much more flexibility to your application. For example, if I wanted to build a mobile app in Android Studio, or an ios app, or multiple web applications or desktop applications, all of these could pull data from a REST api. If I want to use a JavaScript framework on the frontend like React, Angular, or Vue, this now becomes extremely easy and efficient.

This lesson, for your reading, you may continue to use your code base from your e-commerce apps, or you may choose to practice somewhere else as you prepare for your assignments.

Our goals for this lesson are as follows:

  • Learn about RESTful backend applications

  • Compare and contrast server-side rendering vs REST.
