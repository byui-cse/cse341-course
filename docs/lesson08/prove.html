Overview

You will build the backend to an application that already has an existing frontend.

Assignment

  1. The frontend for this project is complete. Please download the frontend project zip file. You will not change any code in it, you will just open the html file in the browser to test that your backend is functional. You'll see a series of ids in the html and the JavaScript. Look at the data that the JavaScript file uses to populate the html, this includes a lot of text, a few links, and a base64 image.

  2. Create a new node project with a REST endpoint to GET all data required by the frontend zip file you downloaded.

  3. While you're writing your code, test it with REST Client in VS Code to ensure it is working correctly.

  4. Once everything looks good with the REST Client, open the frontend webpage in the browser and check to see if it is working (ensure that your backend server is still running on port 8080.

  5. Clean up your REST Client .rest file and add comments for documentation for your backend.

  6. Upload your node project along with the rest file to a new GitHub repository and submit the GitHub link in I-learn.
