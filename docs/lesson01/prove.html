Overview

After completing the assigned reading for the lesson and your team activity, you are prepared to complete your personal assignment. Your assignment will consist of completing the assignment found in Understanding the Basics from the tutorial videos. You will then publish your completed code to a personal GitHub repository. This will help you learn how to work with git, data, pages, and client-server communication.

Instructions

Complete the Assignment - Basics (Problem) from the Academind tutorial by doing the following:

  1. In the main directory of the project template you downloaded during the team activity, create a new file called prove01.js and spin up a Node.js-driven server running on port 3000. (Just ignore the additional folders and files in the project for now.)

  2. Create another file called prove01-routes.js that can handle two routes: "/" and "/users"

    • Return a greeting message on the "/" route

    • Return a list of dummy users on the "/users" route

  3. In the "package.json" file included with the project, add a new startup configuration called "prove01" to the "scripts" section that will run your prove01.js file using nodemon. (Hint: Add "prove01": "nodemon prove01.js" under "scripts")

    You'll need to start your project by typing npm run prove01 in the terminal.

  4. On the "/" page, add a <form> with a "username" <input> that is submitted using a POST request to "/create-user" when the submit button is clicked

  5. Add the "/create-user" route to routes.js and parse the incoming data from the form submission, then display it in the console.

    In order to get 100%, in addition to displaying the submitted data in the console, update the list of users on the /users page.

  6. Be sure to use nodemon

  7. Push your code up to Github.

    8. If you would like help getting started with git this video may be helpful.

  8. Please review rubric below to see how you'll be graded and to ensure you've completed each aspect of this assignment.
