Overview

Assignment

  1. Create an authentication system for your personal project.

  2. Store authentication token in such a way that views are dynamic based on logged in status.

  3. Include user information in the database to allow for future logins.

  4. Please review rubric below to see how you'll be graded and to ensure you've completed each aspect of this assignment.
