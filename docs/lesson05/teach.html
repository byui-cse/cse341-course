Overview

For this lesson's team activity, things will be pretty straight forward. You'll be experimenting with sessions using the MVC architecture we've used so far this semester.

Instructions

Make sure express-session is installed using npm.

  1. In the route path "/" for this Team Activities routes, initiliaze two session variables, "style" and "counter", if they are undefined.

  2. Create a form that utilizes the route "/change-style" and associated controller that will use the session variable "style" to change the the primary color/theme of your .ejs page for this team activity.

  3. Create two forms that will utilize the route "/counter" to increment and decrement the session variable "counter" by a constant of 1.

  4. Finally, create a session that utilizes the route "/reset" to destroy the session.

Tips

  • Remember to redirect to the Team Activity 05's root "/" any time a session variable is modified in order to see the changes.

  • The ability to specify a desired CSS layout (AKA, a theme) using sessions may be useful for a guest user. For example, many websites like YouTube and StackOverflow utilize a "dark mode" that can be used regardless if the user is signed in on their YouTube or StackOverflow account.

  • If a variable used on an EJS page is not specified in the render method, an error message will be displayed.
