  • Team Requirements

  • Team Roles

While working on your first project, you used git to track changes and push your code to Heroku. Now that you're working with a team, you'll have to do a little more than just add, commit, and push. You'll likely want to work on different features on their own branches, and merge finished features into your main branch. Because you're working on a team, you'll also have to deal with merge conflicts, and need to know how to resolve them.

Git is a valuable skill that will be useful to you in other classes, as well as in internships and jobs in the future. To help you gain a better understanding of how git works and how to use it, here are some resources about using git and GitHub:

  • Learn Git Branching - An interactive tutorial for using git on the command line

  • How to use Git and Github (12:18) - Intro to git/GitHub by Fireship on YouTube

  • Git & GitHub Crash Course For Beginners (32:14) - Crash course by Traversy Media on YouTube

  • GitLens - VSCode extension that adds more git integration

Teams participating in this project will also have a team from Frontend Web Development II - WDD 330. In this project, teams in both courses will have to communicate with each other to achieve a common goal. Students in CSE 341 will be responsible for the backend, and students in WDD 330 will be responsible for the frontend.

Project architecture will vary from what was learned in the first part of the semester in some ways. For example, students will deploy a server-side application that will allow another project to retrieve data. Helpful tips and information are including below in Project 2 Resources. It is very common in industry to see teams of frontend developers and teams of backend developers writing code in two different repositories, but that pertain to the same web application as a whole.

Overall, it will be a full stack application. However, the frontend of the application will be built by WDD330 students (with their own git repository) and the backend will be built by CSE 341 students (with their own git repository).
