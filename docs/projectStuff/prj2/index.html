Project 2

Summary

After having created your first project, you are ready to branch out and apply all that you have learned up to this point. For the remainder of the semester, you will work on another project that will account for the majority of your grade in this course. There are two possibilities for your second project. Your instructor will choose which of these projects students will do. Details are listed below.

All students regardless of which project they participate in will be in a team in this class and they will be individually responsible for their grade. Information about teams, how individuals in teams will receive grades based on participation, and team roles can be found here:

  • Team Roles

Project Option 1: Full Stack Application

This project will be similar in architecture to the first project you completed this semester. It will be a full stack application and your team in CSE 341 will be responsible for the database, the backend, and the frontend. Requirements for this project along with all other pertinent information can be found here:

  • Fullstack Project 2 Breakdown

Project Option 2: Backend project - teams from two classes

Teams participating in this project will also have a team from Frontend Web Development II - WDD 330. In this project, teams in both courses will have to communicate with each other to achieve a common goal. Students in CSE 341 will be responsible for the backend, and students in WDD 330 will be responsible for the frontend. Project architecture will vary from what was learned in the first part of the semester in some ways. For example, students will deploy a server-side application that will allow another project to retrieve data. It is very common in industry to see teams of frontend developers and teams of backend developers writing code in two different repositories, but that pertain to the same web application as a whole. Overall, it will be a full stack application. However, the frontend of the application will be built by WDD330 students and the backend will be built by CSE 341 students.
