Overview

This assignment is the start of your first semester project, your E-Commerce Application. Your prove assignments for lessons 2-6 will be primarily derived from the Academind Videos. This is intentional and is designed to help you learn how to develop a well-structured application. Lessons 7-13, you will prove your understanding by building a project of your own design.

Your assignment for this lesson will follow the Academind videos very closely. You will build an express app that handles several routes. If you just follow along with the videos successfully, then you'll be eligible for 93% on this project. Below, there are some ideas for how you could go above and beyond in order to receive 100%.

Assignment

Your tasks are as follows:

  1. Use GET to display a page with all books. (When this page first loads, no products will be present).

  2. Include a link to another page called Add Product. This page should be loaded using GET as well. Include a form on this add product page that will submit a POST request when submitted. This form should include at least the title of the new book. When the form is submitted, add the new book to the array of products and go to the original page which shows all products.

Stretch Challenges

  • Allow user save multiple entries (for example: title, price, description, ratings, etc.).

  • Allow a way for the user to delete a product.

  • Anything else you can think of to make this app more productive. Amazon has implemented a lot of good ideas. :)

Submission

  1. Please review rubric below to see how you'll be graded and to ensure you've completed each aspect of this assignment.

  2. Push code to Heroku (Seeing that this is the start of your E-Commerce App, perhaps you'd like to make a new git repository for it, this is up to you).

    3. Getting Heroku set up with a project can be difficult at first. If you are having issues, this video may help.
