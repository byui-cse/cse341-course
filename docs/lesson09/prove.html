Overview

For this week’s activity, you’ll work on pulling from a 3rd party API using async requests. The API you will be using is pokeapi.co, a popular RESTful API for Pokemon information. JavaScript’s “fetch” method will be used to make the async requests. For now, you’ll only be using GET for the request method.

Assignment

Here are your main tasks:

  1. Populate an unordered list of the first 10 Pokémon using the url https://pokeapi.co/api/v2/pokemon?offset=0&limit=10 for your fetch request.

    NOTE: Clicking the above link will lead you to a JSON output of the API.

  2. Create a button which clears the list and populates it with the next paginated result (this url is identified by the “next” key in the responding JSON object)

  3. Create another button which clears the list and populates it with the previous paginated result. (like next, this url is the value of the “previous key in the responding JSON object)
